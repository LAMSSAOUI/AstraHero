This Project is Built with  (React + vite) and awesome Library  Material Ui only for desktop 

to see the Project :  

  1. clone it in your Desktop :
  
              git clone <Repo_URL>

  2. install dependancies : 
   
              npm install 
              
  3. start with the command : 
  
              npm run dev
